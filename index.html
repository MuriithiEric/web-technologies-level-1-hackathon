Welcome to the Web Technologies Hackathon!

This event is designed to test your skills in HTML elements, including HTML structure, paragraphs, links, images, forms, and CSS. Whether you're a total beginner or looking to brush up on your skills, this hackathon will provide you with an opportunity to practice and showcase your abilities.

Activities:

  • HTML Structure: Create the basic structure of an HTML document.
  • Paragraphs & Headings: Understand how to structure and format text using paragraph tags.
  • Links: Learn how to create hyperlinks to navigate between pages.
  • Images: Incorporate images into your web page.
  • Forms: Understand how to create forms for user input.
  • CSS Styling: Apply styles to your HTML elements using CSS.
